## Project 1


### Snake, Water, Gun Game
A classic game of chance where you play against the computer. Choose snake, water, or gun and see if you can beat the computer!

**How to run:**
```bash
python main.py
```

**How to play:**
- Enter your choice: `s` for snake, `w` for water, or `g` for gun
- The computer makes a random choice
- Winner is determined by the classic game rules:
  - Snake drinks water (snake beats water)
  - Water douses gun (water beats gun)
  - Gun kills snake (gun beats snake)
